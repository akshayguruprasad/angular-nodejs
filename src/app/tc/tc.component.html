The challenge is designed to identify the best programmers in campus and working professionals across the industry.

CONTEST TERMS AND CONDITIONS

  • 1) CODING CONTEST:

    • Languages Accepted: Users can solve the challenge by writing a code in any of the available programming languages depending on which skill you are attempting. The contest will also contain multiple-choice questions (MCQs).

    • Levels in Coding Contest: There will be 3 levels for the coding contest. The first two levels will be launched simultaneously. The final level (3rd) will be a non-premise event launched at a specific date and time. Top 5 coders from the Leaderboard will travel to the Capgemini Mumbai campus to participate in the Finale. Coding problem & MCQs will be timed.The time taken per level will be used to calculate your position on the Leaderboard.

    • Here are the rules for the Coding contest:

    • Level - 1: MCQ

      • There will be 25 MCQs in Level - 1
      • Correct answers for each MCQ will give you 4 points
      • Maximum time allotted for 25 MCQs is 30 minutes
      • Each user will get 2 attempts per level to improve their scores. The higher score from both the attempts will be considered as your final score for that level.
      • Minimum score of 40% is required to move to Level 2.
      • There will be no negative marking in any of the levels.

    • Level 2

      • MCQs
        • For MCQs in Level 2:
        • There will be 25 MCQs in Level 2
        • Correct answers for each MCQ will give you 4 points
        • Maximum time allotted for 25 MCQs is 30minutes
      • Coding
        • For Coding in Level 2:
        • You will get 24 hours for 1 coding challenge. Once you start the level you will have to submit your final code within 24hours
        • Top 5 scorers from Level 2 will qualify for the Finale
        • There will be 1 Coding Problem which will have 10 Test Cases of different complexity:
          • Default test case
          • Basic level test case
          • Boundary level test case
          • Correctness test case
          • Complex test case.
    • Each user will get 2 attempts per level to improve their scores. The higher score from both the attempts will be considered as your final score for that level.
    • Each correct test case while solving the coding challenge will fetch the user 10 points & 4 points for each correct MCQ.
    • There will be no negative marking in any of the levels.
    • Total score will be the cumulative score of all the 2 levels in each contest i.e. Level 1+Level 2.
    • Scoring Logic of Level X is as follows:
      • Number of MCQs Correctly Answered x 4 = A
      • Total Number of Test Cases Answered Correctly x 10 = B
      • Total Score = A + B
    Here are some guidelines that we will follow in the event of a tie-break in Leaderboard positions for Coding Contests:
    1. The user with highest marks (sum of both levels) will be ranked higher.
    2. This being equal, the user with higher marks in the coding problem will be ranked higher
    3. Point 2 being equal, the user with the higher number of correct answers in the MCQ will be ranked higher.
    4. Point 3 being equal, the user who has taken lesser time to answer the MCQs will be ranked higher.
    5. Point 4 being equal, user with the lesser time taken to compile the problem will be ranked higher.
    6. In case of all the top 5 being equal, points 1 through 5 will be repeated with Level 2 and later with Level 1.
    7. Everything being equal, the referral scores of the 2 candidates will be used to sort out the tie break.
    8. In case this too is equal, the user who participated in the contest at the earlier date will be ranked higher.
    Gratifications for Coding contest:
    • 1st Prize - Cash Prize worth INR 30,000/-
    • 2nd Prize - Cash Prize worth INR 20,000/-
    • Early bird prizes (For 1st 10 players to complete Level 2) win Gift Vouchers of INR 500/-each
    • Gift Vouchers will be redeemable from the supplier from where they have been bought

  • 2) Tech Aptitude - THIS IS AN ONLINE CONTEST

    • There will be 3 levels in the above mentioned contest. First 2 levels will be launched simultaneously and 3rd level will be launched on the day of the Finale.
    • There will be 25 MCQs in Level - 1 &2
    • Correct answer for each MCQ will give you 4 points
    • Maximum time allotted for each level is 30minutes
    • A user has to answer 40% of the questions correctly to unlock the next level.
    • Each user will get 2 attempts per level to improve their scores. The highest score from both the attempts will be considered as your final score for that level.
    • There will be no negative marking in any of the levels.
    • The final level for these contests will be held online.
    • Winners of Tech Aptitude Contest will not qualify for the Direct L2 interview.
    Tie Break Logic
    1. The user with highest marks (sum of both levels) will be ranked higher.
    2. Point 1 being equal, the user who has taken lesser time to answer the MCQs of level 3 will be ranked higher.
    3. Point 2 being equal, the user who has taken lesser time to answer the MCQs of level 2 will be ranked higher.
    4. If the situation of tie-break persists even after points 1-3, the user who has taken lesser time to answer the MCQs will be ranked higher.
    5. If the situation of tie-break persists even after points 1-4, the user with the lesser time taken to compile the problem will be ranked higher.
    6. In case this too is equal, the user who participated in the contest at the earlier date will be ranked higher.
